About

My time working in pharmacy has taught me much about attention to detail, time management, and communication. As a Web Developer I seek to use all of these attributes and more to build functional and elegant web applications to meet the needs of any user.

Work

GitHub

With the culmination of the skills I learned at Actualize I built: KickstandsUp. The web application is targeted at fellow motorcycle riders, to give a way to track rides that they are going on and to see other rides going on that they may want to join. KickstandsUp utilizes a Rails backend, PostgreSQL database, a VueJS frontend as well as a few APIs such as Google maps and MapQuest. It also it includes a bit of web scraping with Nokogiri.
